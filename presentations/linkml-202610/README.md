# LinkML for imaging metadata

This directory contains the LaTeX source code for the presentation about
the use of LinkML for imaging metadata at the LinkML workshop planned
for October 2026.


## Copying

The contents of this repository are released under the terms of the
Creative Commons Attribution-ShareAlike 4.0 International License.


## Building

Run `make all` to build the presentation.

Pre-requisites:

* A TeX distribution (tested with [TexLive 2025](https://tug.org/texlive/))
* [Inkscape](https://inkscape.org).
