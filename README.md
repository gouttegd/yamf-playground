YAMF playground
===============

This repository is a playground to experiment with various ways of
dealing with additional data when working with a data model that is
intended to be extensible.

Experiments in this repository are particularly interested in two
different issues:

* How to _store_ the additional data on disk;
* How to _present_ the data (both from the original model, and from any
  extension) to client code.

Ideas explored here may or may not be good, and may or may not end up
being used in a future project.

Use any idea from this repository at your own risk.
